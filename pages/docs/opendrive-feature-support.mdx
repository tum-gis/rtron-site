# OpenDRIVE Feature Support

In the following a list of OpenDRIVE features supported by r:trån is provided.
Currently, there is tested support for OpenDRIVE version 1.4, although other versions can also be read and processed experimentally.

### Geometry

- straight lines <span style={{"color": "green"}}>⬤</span>
- spirals <span style={{"color": "green"}}>⬤</span>
- arcs <span style={{"color": "green"}}>⬤</span>
- cubic polynomials <span style={{"color": "green"}}>⬤</span> (deprecated by ASAM)
- parametric cubic polynomials <span style={{"color": "green"}}>⬤</span>

### Roads

- road linkage <span style={{"color": "green"}}>⬤</span>
- road type <span style={{"color": "green"}}>⬤</span>
- methods of elevation <span style={{"color": "green"}}>⬤</span>
- road elevation <span style={{"color": "green"}}>⬤</span>
- super elevation <span style={{"color": "green"}}>⬤</span>
- road shape <span style={{"color": "green"}}>⬤</span>
- road surface <span style={{"color": "red"}}>⬤</span>

### Lanes

The lane topology of OpenDRIVE is converted to the new Transportation module of CityGML 3.0.
This enables routing on the basis of the predecessor and successor relations of the different traffic spaces.

- lane sections <span style={{"color": "green"}}>⬤</span>
- lane offset <span style={{"color": "green"}}>⬤</span>
- lane linkage <span style={{"color": "green"}}>⬤</span>
- lane properties <span style={{"color": "green"}}>⬤</span>
- road markings <span style={{"color": "green"}}>⬤</span>

### Junctions

- connecting roads <span style={{"color": "green"}}>⬤</span>

### Objects

Depending on the availability and characteristic of the parametric OpenDRIVE geometries, [B-Rep](https://en.wikipedia.org/wiki/Boundary_representation) representations are generated.
Thereby, several geometry corrections are applied (removal of invalid vertices, consecutively following vertex duplicates, ...) and then mapped onto the rich geometry model of [GML](https://www.ogc.org/standards/gml).

- repeating objects <span style={{"color": "green"}}>⬤</span>
- object outlines <span style={{"color": "green"}}>⬤</span>
  - corner roads <span style={{"color": "green"}}>⬤</span>
  - corner locals <span style={{"color": "green"}}>⬤</span>
- object material <span style={{"color": "green"}}>⬤</span>
- lane validity for objects <span style={{"color": "red"}}>⬤</span>
- access rules to parking spaces <span style={{"color": "red"}}>⬤</span>
- object marking <span style={{"color": "red"}}>⬤</span>
- object border <span style={{"color": "red"}}>⬤</span>
- object reference <span style={{"color": "red"}}>⬤</span>
- tunnels <span style={{"color": "red"}}>⬤</span>
- bridges <span style={{"color": "red"}}>⬤</span>
